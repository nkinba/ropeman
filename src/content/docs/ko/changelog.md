Ropeman의 최근 사용자 대상 변경사항을 정리했습니다.

## 2026-04-03

Help 모달, 온보딩 투어, 테스트 커버리지에 집중한 UX 스프린트입니다.

- **Help/Docs 모달** — 사용법 · AI 모드 · 단축키 · FAQ 네 개 탭으로 구성. `?` 키 또는 헤더 버튼으로 열립니다.
- **온보딩 투어** — 2단계 인터랙티브 투어. Prev/Next/Skip 버튼과 화살표 키 지원, Settings에서 재시작 가능.
- **탭 파일명 원본 표기** — 실제 파일명을 대문자 변환 없이 그대로 표시합니다.
- **코드 에디터 폰트 크기 설정** — 10~24 px 슬라이더로 실시간 조정.
- **Settings 모달 재설계** — AI 연결 상태 배지, 캐시 지우기 버튼, 2열 그리드 언어/테마 선택.
- **파일 유형별 아이콘** — Material Symbols + 확장자별 컬러 매핑.
- **시맨틱 브레드크럼 개선** — 각 크럼 클릭 시 자식 노드 드롭다운, 분할 뷰에서도 항상 시맨틱 경로가 보입니다.
- **용어 순화** — "스켈레톤"을 **코드 구조 요약**으로 통일, 기본 크기 150 KB → 250 KB 상향.

## 2026-04-01

MVP 최종 배포 스프린트입니다.

- **ropeman.dev 정식 배포** — 커스텀 도메인으로 프로덕션 배포.
- **탭/패인 UX 개선** — 드릴다운 시 기존 다이어그램 탭 재사용, 코드 탭이 반대 패인에 열려 분할 뷰를 자동 활용.
- **라이트 테마 히어로 색상 보정**.
- **단축키 개선** — `Ctrl+Shift+V`를 사이드바 패널 전환(파일/시맨틱)으로 변경.
- **랜딩 리디자인** — 히어로 타이틀 액센트, 언어 아이콘 축소 배치, Multi-Track AI 카드 추가.

## 2026-03-31

로딩/프로그레스 UX 통일과 WebGPU 캐싱 검증을 다룬 스프린트입니다.

- **로딩 UX 통일** — 글래스모피즘 카드 + 이중 스피너 + Material 아이콘. 분석 진행 상태 StatusBar 추가.
- **WebGPU 캐싱 검증** — 모델 가중치가 브라우저에 정상 캐시되고, 재방문 시 즉시 복원되는 것을 확인.
- **분석 결과 캐시 안정화** — 새로고침 후 AI 재호출 없이 결과가 복원되도록 개선.
- **시맨틱 노드 강조 시각화** — 선택된 노드를 강조하고 나머지는 dim 처리.
- **라이트 테마 보정** — 누락된 CSS 변수 보강, 하드코딩된 색상 값 정리.

## 2026-03-30 (디자인 표준화)

"Obsidian Architect" 디자인을 코드에 전수 반영한 스프린트입니다.

- **공통 컴포넌트 표준화** — 글로벌 공통 클래스 17개 정의, 모든 모달을 공통 클래스로 재작성.
- **레이아웃 정밀 조정** — Header 48 px 고정, Sidebar 48 px 아이콘 바 + 콘텐츠 패널, Canvas 배경 톤 재조정.
- **시맨틱 노드 재설계** — 256 px 폭, 좌측 액센트 바, 엣지 dashed 패턴 + 글로우.
- **빈 상태 디자인** — 큰 헤드라인, SVG 대시 드롭존, 언어 pill 재디자인.
- **Material Symbols 전면 도입** — 이모지 아이콘 제거.
- **Symbol Sidebar 토글** 추가(기본 OFF).

## 2026-03-30 (디자인 시스템 도입)

디자인 시스템의 기초를 다진 스프린트입니다.

- **Obsidian Architect 팔레트** — 7단계 surface 계층, 배경 톤 차이로 영역을 구분(라인 미사용), Space Grotesk + Inter + JetBrains Mono 폰트 조합.
- **글래스모피즘 모달** — 모든 모달에 backdrop-blur 적용.
- **시맨틱 노드 액센트 바** — surface-container-high 배경 + 좌측 컬러 액센트 바.
- **상태 갱신 안정화** — 시맨틱 캐시 갱신 시 항상 즉시 화면에 반영되도록 수정.
