탐색과 분석을 빠르게 만들어 주는 **전체 키보드 단축키**입니다. Mac 사용자는 `Ctrl` 대신 `Cmd`를 사용하세요.

## 탐색

| 동작                             | Windows / Linux        | macOS                 |
| -------------------------------- | ---------------------- | --------------------- |
| 검색 열기                        | `Ctrl` + `K`           | `Cmd` + `K`           |
| 사이드바 토글                    | `Ctrl` + `B`           | `Cmd` + `B`           |
| 사이드바 패널 전환 (파일/시맨틱) | `Ctrl` + `Shift` + `V` | `Cmd` + `Shift` + `V` |
| 다음/이전 탭                     | `Ctrl` + `Tab`         | `Cmd` + `Tab`         |
| 탭 닫기                          | `Ctrl` + `W`           | `Cmd` + `W`           |

## 분할 뷰와 패인

| 동작             | Windows / Linux | macOS       |
| ---------------- | --------------- | ----------- |
| 분할 뷰 전환     | `Ctrl` + `\`    | `Cmd` + `\` |
| 주 패인 포커스   | `Ctrl` + `1`    | `Cmd` + `1` |
| 보조 패인 포커스 | `Ctrl` + `2`    | `Cmd` + `2` |

## 분석과 내보내기

| 동작                | Windows / Linux        | macOS                 |
| ------------------- | ---------------------- | --------------------- |
| 다이어그램 내보내기 | `Ctrl` + `Shift` + `E` | `Cmd` + `Shift` + `E` |

## UI와 도움말

| 동작                    | Windows / Linux        | macOS                 |
| ----------------------- | ---------------------- | --------------------- |
| 테마 전환 (다크/라이트) | `Ctrl` + `Shift` + `D` | `Cmd` + `Shift` + `D` |
| 도움말 열기             | `?`                    | `?`                   |
| 모달 닫기               | `Esc`                  | `Esc`                 |

## 마우스 동작

키보드는 아니지만 자주 쓰는 인터랙션입니다.

- **노드 클릭** — 선택 및 상세 정보 확인
- **노드 더블클릭** — 드릴다운 탐색 (하위 구조 분석)
- **파일 더블클릭** — 코드 뷰어에서 열기
- **탭 드래그** — 반대 패인으로 이동해 분할 뷰 생성

단축키가 동작하지 않는다면 브라우저 확장 프로그램이 동일한 조합을 가로채고 있지 않은지 확인해 주세요.
